BIG FIVE CORRESPONDENCE

IPIP（International Personality Item Pool）を参考に、ビッグファイブの5因子をエンジニア文脈の4軸+2補助に対応付け。

EXTRAVERSION（外向性）
→ 思考スタイル軸 (D/T)
← Deep Focus（内向・集中）Talk-through（外向・対話）→
外向性が低い人は一人で深く考えることを好み、高い人は対話を通じて思考を整理する傾向がある。IPIPの「社交性」「積極性」因子に対応。
OPENNESS TO EXPERIENCE（開放性）
→ 情報処理軸 (B/A)
← Builder（具体・実装）Architect（抽象・設計）→
開放性が低い人は具体的・実践的なアプローチを好み、高い人は抽象的概念やアーキテクチャ思考を好む。IPIPの「知的好奇心」「想像力」因子に対応。
EMOTIONAL STABILITY + OPENNESS
→ リスク姿勢軸 (S/C)
← Steady（安定・慎重）Challenger（挑戦・革新）→
情緒安定性が高く開放性が低い人は安定を重視し、逆の組み合わせは新しい挑戦を好む。技術選定やデプロイ判断に特に表れる。
CONSCIENTIOUSNESS（誠実性）
→ 作業様式軸 (P/X)
← Planner（計画・体系）eXplorer（探索・即興）→
誠実性が高い人は計画を立て体系的に進めることを好み、低い人は柔軟に探索しながら進める。IPIPの「秩序性」「自己規律」因子に対応。
AGREEABLENESS（協調性）
→ 協働スコア（補助）
← 技術的正しさ重視共感・関係性重視 →
コードレビューやチーム内議論での姿勢に表れる。タイプコードには含まれず、補助スコアとして記録。
OPENNESS（開放性の別側面）
→ 学習スコア（補助）
← Specialist（深掘り）Generalist（幅広）→
技術習得の方向性。深さ vs 広さの志向を測定。タイプコードには含まれず、補助スコアとして記録。
